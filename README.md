# Nutty - Backend

## Description

Nutrition tracker backend built with NestJS

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ docker-compose pull
$ npm run docker:start
# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
